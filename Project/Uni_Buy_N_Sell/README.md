# Uni Buy N Sell - Campus Marketplace

A modern web-based marketplace platform designed for university students to buy and sell items within their campus community. Built with HTML, CSS, and JavaScript, inspired by OLX and Depop.

## 🎯 Project Overview

Uni Buy N Sell is a semester project that provides a trusted platform for students to trade items such as textbooks, electronics, furniture, fashion items, and more. The platform emphasizes safety, community, and ease of use.

## ✨ Features

### Landing Page (index.html)
- **Hero Section**: Engaging introduction with search functionality
- **Statistics Display**: Shows active listings, users, and categories
- **Categories Grid**: Browse 8+ major categories
- **How It Works**: Step-by-step guide for new users
- **Call to Action**: Encourages user registration
- **Responsive Footer**: Links and social media integration

### Home/Browse Page (home.html)
- **Advanced Search Bar**: Search with filters for category and location
- **Sidebar Filters**: 
  - Price range selection
  - Condition filters (New, Like New, Good, Fair)
  - Seller type filters
- **Product Grid**: Display of 12 sample listings with:
  - Product images
  - Pricing
  - Location and time posted
  - Seller information
  - Favorite button
- **Sorting Options**: Sort by newest, price (low to high, high to low), and popularity
- **View Toggle**: Switch between grid and list view
- **Pagination**: Navigate through multiple pages of results

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS)
- **Icons**: Font Awesome 6.4.0
- **Design Inspiration**: OLX and Depop

## 🚀 Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- No installation required

### Running the Project

1. Clone or download the project files
2. Open `index.html` in your web browser for the landing page
3. Navigate to `home.html` for the browse/listings page
4. No server setup required - runs entirely in the browser

### Opening Files
```bash
# Navigate to project directory
cd Uni_Buy_N_Sell

# Open in browser (Windows)
start index.html

# Or simply double-click index.html in File Explorer
```
